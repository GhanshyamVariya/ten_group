# ten_group

* Clone this repo.:
    ```bash
    $ gh repo clone GhanshyamVariya/ten_group
    $ cd ten_group
    $ pip install -r requrements.txt
    ```

#

*  Execute migrations file:
    ```bash
    $ python manage.py migrate
    ```

#

*  Run the server locally at __*http://localhost:8000/*__ :
    ```bash
    $ python manage.py runserver 8000
    ```
